About Project
INEVITABLE — Smart Water Monitoring System

Project Overview

INEVITABLE is an IoT-based smart water monitoring and management system that provides real-time insights into water usage, automated motor control, and intelligent safety systems — all accessible from any device.

Core Features

Real-time water level monitoring, automated overflow/dry-run protection, AI-driven usage forecasting, multi-tank management, SMS/email alerts, and a fully responsive dashboard.

Hardware Stack

Arduino Uno · HC-SR04 Ultrasonic Sensor · YF-S201 Flow Sensor · L298N Motor Driver · ESP8266 WiFi Module

Technology Stack

HTML5CSS3JavaScript ReactNode.jsFirebase Chart.jsTwilioArduino IDE ThingSpeakTensorFlowVercel

Safety Systems

Overflow Prevention · Dry Run Protection · Leak Detection · Auto Shutdown · Overflow Alert · Auto Restart after Power Cut
